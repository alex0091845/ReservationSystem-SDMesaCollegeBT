# ReservationSystem-SDMesaCollegeBT
The reservation system project for at San Diego Mesa College for the Business and Technology (BT) building built by the computer science students attending Mesa College!

To test the web server, go into the backend folder on your terminal:
`cd backend`

And then run the Spring Boot application with the mock profile:
`mvn spring-boot:run -Dspring-boot.run.profiles=mock`
